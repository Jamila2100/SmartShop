# SmartShop 🛍️

A price comparison and shop filtering app that helps users find the best products based on price, delivery time, and ratings.

## Tech Stack
- FastAPI (Backend)
- React Native / Expo (Frontend)
- Web Scraping

## Features
- Compare products across platforms
- Filter based on price, rating, delivery time
- Suggest best matched products

## How to Run

### Backend
cd Shop Filter Backend
python -m uvicorn main:app --reload

cd Pricewise Backend Main
python -m uvicorn main:app --port 8001 --reload

### Frontend
cd SmartShop Frontend Main
npx expo start
